# Strand Sort Desc-
This program implements the Strand Sort algorithm in descending order using C++. The algorithm works by repeatedly extracting a sublist from the input list. Each strand contains elements that are in descending order.

The program starts by taking the first element and then compares it with the rest of the elements, adding those that satisfy the descending condition. These elements are removed from the original list and added to the strand.

Each strand is then merged into a result list while maintaining descending order. This process continues until the input list becomes empty.

The program also keeps track of the number of comparisons and moves performed during sorting to analyze the algorithm’s efficiency.
